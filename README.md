**Warning**: don't write command from plugin - it's kill all files and OS on your pc

installing:
1. copy this folder in ~/.config/quickshell
2. add in JES config:
```toml
[plugin.activate-linux]
source = "activate linux meme/Main.qml"
active = true
```
